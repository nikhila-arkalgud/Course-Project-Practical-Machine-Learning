Introduction

Using devices such as Jawbone Up, Nike FuelBand, and Fitbit it is now possible to collect a large amount of data about personal activity relatively inexpensively. These type of devices are part of the quantified self movement – a group of enthusiasts who take measurements about themselves regularly to improve their health, to find patterns in their behavior, or because they are tech geeks. One thing that people regularly do is quantify how much of a particular activity they do, but they rarely quantify how well they do it. In this project, your goal will be to use data from accelerometers on the belt, forearm, arm, and dumbell of 6 participants. They were asked to perform barbell lifts correctly and incorrectly in 5 different ways. More information is available from the website here: http://groupware.les.inf.puc-rio.br/har (see the section on the Weight Lifting Exercise Dataset). http://groupware.les.inf.puc-rio.br/har

Data Preparation

We will load the data and the 20 cases that will be submitted to coursera. Download the pml-training.csv and pml-testing.csv files from http://d396qusza40orc.cloudfront.net/predmachlearn

pmltesting <- read.csv("http://d396qusza40orc.cloudfront.net/predmachlearn/pml-testing.csv", sep=",", na.strings=c("","NA"))

Data Cleaning

We will remove columns with NAs and only use features used in the test set

pmltesting1 <- pmltesting[1:159] # temporarily remove last column problem_id from test set
features <- names(pmltesting1[,colSums(is.na(pmltesting1)) == 0]) # check for NA
pmltraining <- pmltraining[,c(features,"classe")] # select only non NA features and include classe column
pmltesting <- pmltesting[,c(features,"problem_id")] # select only non NA features and include problem_id column

We will use divide training data set into 75% to train and 25% to test to fit a model.

Feature Selection

We will remove the high correlation features > 90%

outcome <- which(names(train)=="classe")
train_numeric <- train[sapply(train,is.numeric)]
highCorrCols <- findCorrelation(abs(cor(train_numeric[,-outcome])),0.9)
highCorFea <- names(train[highCorrCols]) 
train <- train[,-highCorrCols] # remove the high correlation features
